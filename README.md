# CLI Cheatsheet

1. `cd [folder name]` -> change directory
2. `md [folder name]` -> make directory
3. `code [folder name]` -> vs code CLI command
4. `explorer [folder name]` -> windows file explorer
5. `.` -> current directory
6. `..` -> back directory
7. `ls` -> list directory items
8. `clear` -> clear the console/terminal
9. `new-item [file name]` -> create new file

# GIT Cheatsheet

1. `git init` -> Initialize a new repository
2. `git add [file name]` -> add file to git tracking
3. `git commit -am [commit message]` -> commit file changes
4. `git log` -> show full log
5. `git log --oneline` -> show one line log
6. `git reset [commit number]` -> soft travel to previous commit
7. `git reset --soft [commit number]` -> travels to previous commit but keep changes visible
8. `git reset --hard [commit number]` -> travel to previous commit but don't keep changes visible
